# Insurance Fraud Detection Using Machine Learning

An AI-powered system that detects fraudulent insurance claims 
using supervised machine learning models including Random Forest, 
Decision Tree, KNN, and Gradient Boosting.

## Tech Stack
- Python 3.9+
- Scikit-learn
- Pandas, NumPy
- Matplotlib, Seaborn
- Flask
- ngrok (for public deployment)

## Models Used
- Decision Tree
- Random Forest (primary)
- K-Nearest Neighbours
- Gradient Boosting

## Deployment
The Flask app is run locally and exposed publicly via ngrok.
